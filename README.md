The traditional Who's That Pokemon game has become a little too easy for my friends and I to play. That's why I want to create a new version of the game using sprites from Pokemon Rumble! I love how much character is expressed in these balloon-like designs and think it would increase the difficulty of the game while keeping the spirit the same.

Sprites sourced from https://models.spriters-resource.com/3ds/pokemonrumbleworld/
